# MR forest plot

Adaptation from TwoSampleMR::mr_forest_plot()

## Usage

``` r
mr_forest_plot2(singlesnp_results, exponentiate = FALSE, alpha = 0.05)
```

## Arguments

-   singlesnp_results:

    Results based on single variants.

-   exponentiate:

    Logic variable to indicate exponentiation, Default: FALSE.

-   alpha:

    Confidence level, Default: 0.05.

## Value

A graphic object.
