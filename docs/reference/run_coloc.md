# Colocalisation analysis

The function takes eQTL and GWAS summary statistics for a colocalisation
analysis.

## Usage

``` r
run_coloc(eqtl_sumstats, gwas_sumstats, harmonise = TRUE)
```

## Arguments

-   eqtl_sumstats:

    eQTL summary data.

-   gwas_sumstats:

    GWAS summary data.

-   harmonise:

    a flag to harmonise data.

## Value

Summary from `coloc.abf`.
