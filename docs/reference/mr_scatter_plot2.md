# MR scatter plot

Adaptation of TwoSampleMR::mr_scatter_plot()

## Usage

``` r
mr_scatter_plot2(mr_results, dat, alpha = 0.05)
```

## Arguments

-   mr_results:

    MR results.

-   dat:

    data.

-   alpha:

    Confidentce level, Default: 0.05.

## Value

A graphic object.
